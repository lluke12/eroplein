**Datum:** Maart 2026
**Project:** Eroplein - Nederlands erotisch bedrijvengids/reviewplatform
**Scope:** Keyword research, concurrentieanalyse, paginastrategie, technische SEO

## Managementsamenvatting

### 5 Belangrijkste Bevindingen

1. **De markt is enorm maar gefragmenteerd.** kinky.nl domineert met 3.38M maandelijkse bezoeken (top-5 adult site in NL), gevolgd door sexjobs.nl (872K) en girlsreview.nl (582K). Na het verdwijnen van hookers.nl (2021) is er een gat in de markt voor een betrouwbaar, onafhankelijk reviewplatform.

2. **Stad + categorie combinaties zijn de kern van organisch verkeer.** Zoekopdrachten als "escort amsterdam", "privehuis rotterdam", "erotische massage den haag" hebben consistent hoog zoekvolume. Eroplein's bestaande URL-structuur (`/[stad]/[categorie]/`) is perfect gepositioneerd voor deze queries.

3. **Informatieve content is een onderbenutte kans.** Concurrenten focussen bijna uitsluitend op advertenties/listings. Er is nauwelijks kwalitatieve informatieve content over prijzen, regels, ervaringen voor beginners, stadsgidsen etc. Dit is precies waar Google E-E-A-T-signalen zoekt.

4. **Reviews zijn de unieke waardepropositie.** Sinds hookers.nl (87.000 leden, 9M pageviews/maand) is verdwenen, is er geen centraal, betrouwbaar reviewplatform meer. GirlsReview.nl heeft ernstige privacyproblemen en een lage vertrouwensscore. Dit gat kan Eroplein vullen.

5. **Technische SEO en structured data bieden directe winst.** Schema markup (LocalBusiness, AggregateRating, FAQPage, BreadcrumbList) kan de CTR met 20-30% verhogen. De programmatic SEO-aanpak met 336+ landingspagina's is sterk, mits elke pagina unieke, waardevolle content bevat.

### 2.2 kinky.nl - Marktleider

**Verkeer:** 3.38M/maand (85% mobiel, 15% desktop)
**Bounce rate:** 12.74% (uitzonderlijk laag)
**Pagina's per sessie:** 19.5 (zeer hoog engagement)

**Wat ze goed doen:**
- Breed aanbod: escorts, BDSM, massage, fetish, webcam, telefoon
- Geavanceerde zoekfunctie met filters (locatie, leeftijd, diensten, kenmerken)
- Intern berichtensysteem voor veilige communicatie
- Verificatieproces voor adverteerders
- Mobiel-first ontwerp (85% mobiel verkeer)

**Wat ze NIET doen:**
- Geen echte onafhankelijke reviews/ervaringen
- Geen informatieve content/blog
- Geen stadsgidsen of beginner-guides
- Geen vergelijkingspagina's
- Geen FAQ-secties

**Kans voor Eroplein:** kinky.nl is puur een advertentieplatform. Ze missen volledig de review- en informatiekant.

### 2.3 sexjobs.nl - Erotische Marktplaats

**Verkeer:** 872K/maand
**Bounce rate:** 47.55%
**Pagina's per sessie:** 6.01
**Actief sinds:** 2002

**Wat ze goed doen:**
- 150+ subcategorieen, zeer gedetailleerde categorisering
- 3000 nieuwe advertenties per dag
- Sterk merk (branded search "sexjobs" = 450K/maand)
- 70% direct verkeer (sterke merkbekendheid)
- 1.03K referring domains

**Wat ze NIET doen:**
- Geen reviewsysteem
- Geen redactionele content
- Geen stadsgidsen
- Geen educatieve pagina's
- Geen structured data optimalisatie (beperkt)

**Kans voor Eroplein:** sexjobs.nl is een "marktplaats" model. Eroplein kan zich onderscheiden als reviewplatform met redactionele kwaliteit.

### 2.4 girlsreview.nl - Reviewplatform (met problemen)

**Verkeer:** 582K/maand
**Bounce rate:** 22.02%
**Pagina's per sessie:** 10.7

**Problemen:**
- Autoriteit Persoonsgegevens (AP) is onderzoek gestart
- Tientallen klachten van sekswerkers over privacy
- "Waar is deze dame?" functie deelt persoonlijke data zonder toestemming
- Scam Detector trust score: 38.2/100
- Slechte reputatie bij sekswerkers zelf

**Kans voor Eroplein:** GROOT. Er is een duidelijk gat in de markt voor een ethisch, betrouwbaar reviewplatform dat zowel klanten als sekswerkers respecteert. Eroplein kan zich positioneren als het "Trustpilot van de erotische industrie" - transparant, eerlijk, met privacybescherming.

### 2.5 hookers.nl - Verdwenen Marktleider

**Status:** Offline sinds 2021
**Was:** 87.000 leden, 9M pageviews/maand
**Reden sluiting:** Privacy-overwegingen na datalek (250.000 gebruikersgegevens gelekt in 2019)

**Impact op de markt:** De community is verspreid over kleinere forums, Telegram-groepen en Discord-servers. Er is geen centraal platform meer voor reviews en ervaringen. Dit is de grootste kans voor Eroplein.

## 4. Technische SEO Aanbevelingen

### 4.1 Schema Markup (Structured Data)

#### Per paginatype implementeren:

**Homepage - WebSite + Organization:**
```json
{
  "@context": "https://schema.org",
  "@type": "WebSite",
  "name": "Eroplein",
  "url": "https://eroplein.com",
  "description": "Nederlands platform voor reviews en beoordelingen van erotische diensten",
  "potentialAction": {
    "@type": "SearchAction",
    "target": "https://eroplein.com/zoeken?q={search_term_string}",
    "query-input": "required name=search_term_string"
  }
}
```

**Bedrijfspagina - LocalBusiness + AggregateRating:**
```json
{
  "@context": "https://schema.org",
  "@type": "LocalBusiness",
  "name": "Bedrijfsnaam",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "Straat 123",
    "addressLocality": "Amsterdam",
    "addressRegion": "Noord-Holland",
    "postalCode": "1012 AB",
    "addressCountry": "NL"
  },
  "telephone": "+31-20-1234567",
  "priceRange": "$$",
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "4.2",
    "reviewCount": "47",
    "bestRating": "5",
    "worstRating": "1"
  }
}
```

**Reviewpagina - Review:**
```json
{
  "@context": "https://schema.org",
  "@type": "Review",
  "itemReviewed": {
    "@type": "LocalBusiness",
```
